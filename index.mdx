      <Step title="Install the Package">
        Add Centralised UI to your Unity project via the Package Manager using the Git URL or download from the repository.
        
        ```text
        https://github.com/Sadrmm/CentralisedUI.git
        ```
      </Step>
      
      <Step title="Add Component to GameObject">
        Select your UI Selectable (Button, Toggle, Slider, etc.) and add the **CentralisedUIData** component through the Inspector or create a new GameObject with the component pre-added from the Unity Editor menu under **UI > ProgramadorCastellano > CentralisedUI**.
      </Step>
      
      <Step title="Create ScriptableObjects">
        Right-click in your Project window, navigate to **Create > ProgramadorCastellano > CentralisedUI** and choose the transition type you need:
        - **ColorTint** for color-based transitions
        - **SpriteSwap** for sprite-based transitions  
        - **Animation** for animation-based transitions
      </Step>
      
      <Step title="Apply Your Skin">
        Assign your created ScriptableObject to the component, select your transition mode, and click the **Apply Skin** button. Your UI styling will be applied instantly and persist across all selectables using the same ScriptableObject.
      </Step>
